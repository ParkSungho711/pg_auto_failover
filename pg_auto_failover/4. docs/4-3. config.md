<h1>pgautofailover운영에 관련된 설정 파일</h1>

<h3>1. postgresql.conf</h3>
  - postgresql 성능에 직접적으로 영향을 주는 파라미터를 설정하는 공간
  - 일부 파라미터 수정 시 pgautofailover 서비스 재기동 필요함

<h3>2. pg_hba.conf</h3>
  - 각 노드별 통신과 app에서 들어오는 ip 등 접근 권한에 대한 설정

<h3>3. pgautofailover.service</h3>
  - 서비스로 pgautofailover 기동하기 위한 설정
  - 서비스로 기동 시 /var/log/messege 에 기록이 남게됨

<h3>4. pgautofailover.service start</h3>
  - 실제 db 서비스 기동
  - monitor 노드에서는 formation 전체 세트 확인 가능
  - primary, secondary 에서는 현재 pgautofailover Set 상태 확인 가능
